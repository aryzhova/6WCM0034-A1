Use Cases

The following use cases are described, considering that the users installed the app on their phone and logged in.

Use Case 1: Finding a recipe by ingredients

1. User find the option ‘Search’ in the side menu and taps it.

1.1 System opens a page with an input field, an ‘Add’ button, and ‘Show recipes’ button.

2. User types in an ingredient and clicks ‘Add’ button

2.1 The ingredient appears under the input field

2.2 User continues to type all the ingredients they are looking for, until they are done.

3. User clicks ‘Show recipes’ button

3.1 System shows all recipes that match user’s criteria
