Goal of the Final App

The title of the app that will be finished in assignment 2 is Recipe Finder. It represents a collection of recipes. The users of the application will be able to do the following:

  • Create an account
  • Add their own recipes
  • Browse all recipes added by other users
  • Search for recipes by ingredients
  • Save recipes added by other users
  • Add missing ingredients from the recipe profile to their shopping list
  • Browse their shopping list and add more items manually
Users will also be able to add pictures of their recipe dishes, using their device camera. To search for the recipes they will need to type in the ingredients in an input field. Only those recipes will be shown, that contain a subset of the ingredients that the user entered. This comes handy when the user doesn't have time to shop for the missing products.
